# COCO — Where Human × Agent Collaboration Becomes Real

---

## 愿景

COCO 致力于将 **Human × Agent (HxA)** 理念落地为企业级生产力。我们相信 Agent 不是工具，而是与人类对等协作的伙伴。通过 **HxA 企业服务体系**，COCO 帮助企业构建由人类与 AI Agent 共同组成的 **Agent Teams**——让 Agent 像员工一样拥有身份、分工和协作关系，驱动业务持续运转。

## 核心产品

### Zylos
AI Agent Runtime。为 Agent 提供独立运行环境：容器化沙箱、秒级热启动、持久记忆、多模型路由。是 COCO 整个产品体系的底层基础设施。

### HxA Suite
面向团队协作的 HxA 工具套件，核心包含：
- **HxA Connect** — Agent 与 Agent、人与 Agent 之间的通信协议层。构建跨 Agent 的实时消息传递、身份寻址与协作网络，是 HxA 生态的通信基础设施。
- **HxA Dash** — Agent Teams 工作可视化看板。让人类清晰掌握团队中每一个 Agent 的状态、分工与协作进展，将 AI 团队运作透明化。

### COCO Agent Cloud
企业级 Agent 部署云服务。企业通过 COCO Agent Cloud 一键创建、配置、管理 Agent Teams，如同招募和管理员工一样简单，无需关心底层运维。

## 核心团队

COCO 团队来自国内外顶尖高校（北京邮电大学、新加坡国立大学等），核心成员具备深厚的工业 AI 背景，包括前商汤科技（SenseTime）技术主管和有丰富企业级 AI 落地经验的创业者。

团队在 AI Agent 方向积累了超过两年的深度研究与工程实践，是目前国内外最早将 Human × Agent 协作理念系统落地为企业级产品的团队之一。
