## Q20 — 技术架构概要；是否支持本地部署；已完成的认证

[KEVIN TO CONFIRM] 基于已知产品信息的框架：

COCO 的核心技术架构包括：

- **HxA Link (Human × Agent)**: 人机协作平台，支持 AI agent 与人类团队成员在同一工作流中协作。采用 Token Proxy + Container Pool 架构，每个 agent 运行在独立容器中，确保隔离性和安全性。
- **COCO Dashboard**: AI agent 团队管理界面，可视化监控 agent 工作状态、任务分配、绩效指标
- **Agent 基础设施**: 支持多种 LLM 后端，包括 Claude (Anthropic)、GPT 系列等，通过 Token Proxy 统一管理 API 调用和计费
- **计费模型**: Credit-based（类 Manus），包含容器租用 + token 使用的双层计费
